## Описание

Web-приложение, имитирующее работу банковской системы. [file:1]  
Пользователи регистрируются, открывают счета, переводят средства, конвертируют валюты и читают финансовые новости. [file:1]  
Backend — Java (Spring Boot), frontend — React, БД — PostgreSQL. [file:1]

## Функциональность

- Регистрация и авторизация — JWT-токены, роли (пользователь/админ). [file:1]
- Банковские счета — создание, просмотр, закрытие счетов в разных валютах. [file:1]
- Накопительный счёт — начисление процентов по расписанию. [file:1]
- Пополнение и переводы — пополнение счёта, перевод средств между пользователями по номеру счёта. [file:1]
- Конвертация валют — обмен между валютами по актуальному курсу (внешний API). [file:1]
- Настройки аккаунта — смена личных данных, переключение темы интерфейса. [file:1]
- Новостная лента — финансовые новости, акции, текущие курсы валют (внешний API). [file:1]
- Ассистент — чат-бот. [file:1]

## Архитектура

Микросервисная архитектура, минимум 2 сервиса. [file:1]

### Сервисы

| Сервис       | Что делает                                                                 |
|--------------|----------------------------------------------------------------------------|
| Core Service | Пользователи, счета, переводы, авторизация, накопительные счета. [file:1] |
| Info Service | Новостная лента, курсы валют, конвертация, ассистент. [file:1]            |

Взаимодействие между сервисами — REST (синхронно) + Kafka для асинхронных событий. [file:1]  
У каждого сервиса своя БД. [file:1]  
Frontend обращается к сервисам через API. [file:1]
