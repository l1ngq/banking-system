# ТЕХНИЧЕСКОЕ ЗАДАНИЕ

## Проект

Проект: Банковская система (web-приложение)  

Команда: Балыхин Максим (DevOps), Мун Николай (Backend), Боланел Игорь (Frontend)  

## Описание

Web-приложение, имитирующее работу банковской системы.    
Пользователи регистрируются, открывают счета, переводят средства, конвертируют валюты и читают финансовые новости.    
Backend — Java (Spring Boot), frontend — React, БД — PostgreSQL.  

## Функциональность

- Регистрация и авторизация — JWT-токены, роли (пользователь/админ).  
- Банковские счета — создание, просмотр, закрытие счетов в разных валютах.  
- Накопительный счёт — начисление процентов по расписанию.  
- Пополнение и переводы — пополнение счёта, перевод средств между пользователями по номеру счёта.  
- Конвертация валют — обмен между валютами по актуальному курсу (внешний API).  
- Настройки аккаунта — смена личных данных, переключение темы интерфейса.  
- Новостная лента — финансовые новости, акции, текущие курсы валют (внешний API).  
- Ассистент — чат-бот.  

## Архитектура

Микросервисная архитектура.  

### Сервисы

| Сервис       | Что делает                                                                 |
|--------------|----------------------------------------------------------------------------|
| Core Service | Пользователи, счета, переводы, авторизация, накопительные счета.   |
| Info Service | Новостная лента, курсы валют, конвертация, ассистент.              |

Взаимодействие между сервисами — REST (синхронно) + Kafka для асинхронных событий.    
У каждого сервиса своя БД.    
Frontend обращается к сервисам через API.
